# enigma

Basic python project

* Object oriented programming
* Unittest
* Pre-commit
* CI (Github Actions)

## Setup

* pre-commit
  * `touch .pre-commit-config.yaml`
  * edit .pre-commit-config.yaml
  * `pre-commit install`
  * `pre-commit autoupdate`
* github actions
  * `mkdir -p .github/workflow/`
  * `edit .github/workflow/<file_name>.yml`
  * edit <file_name>.yml
